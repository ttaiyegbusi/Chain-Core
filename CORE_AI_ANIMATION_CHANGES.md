# Core AI Animation Pass

This build focuses only on subtle Core AI chat animations and interaction polish.

## Added / Updated

- Message entrance remains a calm fade-up motion.
- Thinking state keeps the soft shimmer + dot pulse.
- Researching state now slides in subtly instead of hard-appearing.
- Chart cards now reveal with a short fade/scale animation.
- Follow-up chips now reveal one-by-one with a slight stagger.
- Composer now has a focus affordance: subtle surface lift and inner input emphasis.
- Send button now has a small active press interaction.
- Expanded panel content now settles after the shell expands.
- Attachments panel now enters with a short opacity/slide motion.
- Copy action now shows a small "Copied" tooltip without shifting layout.
- Reduced-motion accessibility fallback added for new animation classes.

## Scope

No unrelated accounting screens were changed. This pass only touches Core AI animation/interaction files and global Core AI animation styles.
